Hello World

The “Hello world!” is a small code that can be developed in different programming languages.

The objective is to facilitate the knowledge of the basic structure and the functioning of the language.

Therefore, it can be developed in several alternatives. In practice, it is a simple program, with few lines of code and that returns the message “Hello, world!”

On the screen, here are some examples.

import javax.swing.JFrame;  //Importing class JFrame
  import javax.swing.JLabel;  //Importing class JLabel
  public class HelloWorld {
      public static void main(String[] args) {
          JFrame frame = new JFrame();           //Creating frame
          frame.setTitle("Hi!");                 //Setting title frame
          frame.add(new JLabel("Hello, world!"));//Adding text to frame
          frame.pack();                          //Setting size to smallest
          frame.setLocationRelativeTo(null);     //Centering frame
          frame.setVisible(true);                //Showing frame
      }
  }
